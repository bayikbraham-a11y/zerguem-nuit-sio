Bonjour,

Je vous soumets notre projet pour le défi "Zerguèm de la nuit".

Lien du jeu : [https://bayikbraham-a11y.github.io/zerguem-nuit-sio/]

Procédure pour activer la partie :
1. Ouvrir le lien ci-dessus
2. Configurer le nombre de joueurs (1 à 10)
3. Saisir les noms des joueurs
4. Cliquer sur "Démarrer la partie"
5. Utiliser la souris pour viser et cliquer pour tirer
6. Le jeu dure 45 secondes par joueur

Un bouton "Aide & Instructions" est disponible sur l'écran d'accueil
pour plus de détails sur les phases de jeu et le système de points.

Fonctionnalités implémentées :
- Configuration de 1 à 10 joueurs
- 3 phases de jeu avec évolution des ennemis
- Système de score avec précision
- Écran de transition entre joueurs
- Tableau des scores final
- Design responsive et effets visuels
